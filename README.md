## Como usar

1. Instale os _node modules_

    ```bash
    npm install
    ```

2. Inicie o serviço do [GitBook](https://goo.gl/Y94U1W)

    ```bash
    npm start
    ```
    > Por default o serviço será iniciado em localhost:4000
